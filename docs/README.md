# Smart Campus Management System (SCMS)

**HITEC University, Taxila**

This folder contains project documentation and branding assets for the HITEC
University Smart Campus Management System.

## Branding Assets

- `logo.svg`: final selected vector logo.
- `logo.png`: rendered PNG logo for documents and previews.
- `logo.ico`: Windows icon with 16, 32, 64, 128, and 256 px entries.
- `logo_variant_campus.svg`: campus building variant.
- `logo_variant_cap.svg`: graduation cap variant.
- `logo_variant_minimal.svg`: minimal shield variant.
- `cover_page.md`: Markdown cover page source.
- `cover_page.pdf`: rendered cover page with the logo.

## Notes

The logo uses a shield, campus building, and graduation cap to represent a
university-focused smart campus system. The Windows folder icon setup is local
machine-specific, so the portable submission keeps the icon file in `docs/`.
